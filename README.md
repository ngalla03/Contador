# Contador

Aplicación básica desarrollada en **Android Studio** usando el lenguaje **Kotlin**,, su funcionamiento es un sencillo contador.
El objetivo del proyecto es demostrar el uso de layouts, buttons y estados.

## Características principales

- Posibilidad de incrementar o decrementar de **1 en 1** o de **5 en 5**.
- Bloqueo para evitar numeros negativos.
- Mensajes de error.
- Contador de todas las interacciones.
- Botón para **borrar** todos los datos.
- Conservación del estado al rotar la pantalla gracias a **rememberSaveable**.
